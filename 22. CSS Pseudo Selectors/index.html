Pseudo Selectors

Pseudo-Class Selectors

    Target elements in a specific state or position.
  • :hover: Targets an element when the mouse pointer is over it. Example: a:hover { text-decoration: underline; }
  • :focus: Targets an element when it has focus. Example: input:focus { border-color: blue; }
  • :nth-child(n): Targets the nth child of its parent. Example: li:nth-child(odd) { background-color: grey; }

Pseudo-Element Selectors

    Target part of an element's content in a way that's not possible with other selectors.
  • ::before: Inserts content before an element's content. Example: p::before { content: "Note: "; font-weight: bold; }
  • ::after: Inserts content after an element's content. Example: p::after { content: ";"; }
